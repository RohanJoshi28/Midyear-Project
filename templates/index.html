Emotion Detector Project

Rohan Joshi and Christopher Rodriguez

The goal of this project is to classify emotions into 7 main classes (some of the most common emotions). Once you enter an image, a convolutional neural network will classify your image into one of these classes. This project uses the FER2013 dataset to train (here) so please put the images in that format if you hope to obtain accurate results when uploading. Otherwise, you can wait a couple moments for your webcam to turn on for live emotion detection and classification.
